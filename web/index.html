HTML

HTML no es un lenguaje de programaciÓn como tal porque no tiene estructura de control, no tiene condicionales, no tiene bucles no tiene nada,

simplemente es un lenguaje de marcado o de etiquetado que nos permite crear la estructura básica el esqueleto básico de una página web.

Todo el codigo HTML esta conformado por etiquetas, una etiqueta simplemente
